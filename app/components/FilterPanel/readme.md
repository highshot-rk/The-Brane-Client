## Object shapes

Tag:
```js
const tag = {
  _key: '12345',
  title: 'Cluster Name',
  children: TagTree
}
```

TagTree:
```js

const tagTree = [Tag, Tag, Tag]
```

NodeFilter and TagPath:
```js
const nodeFilter = [{
  _key: '12345',
  title: 'Cluster Name'
}, {
  _key: '12345',
  title: 'Child Cluster Name'
}, {
  _key: '12345',
  title: 'Child Child Cluster Name'
}]
```
